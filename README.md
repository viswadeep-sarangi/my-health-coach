# Health Coach App - Final Clean Build

This package serves the final clean build:
- premium wellness styling
- onboarding flow
- full journey dashboard
- no sticky dashboard overlap
- ready to build and run with Docker

## Run with Docker
```bash
docker build -t health-coach-app .
docker run --rm -p 8080:80 health-coach-app
```

Open:
http://localhost:8080

## Run with Docker Compose
```bash
docker compose up --build
```

Open:
http://localhost:8080
